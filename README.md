# Bookmark manager

## Getting started

    git clone https://github.com/merima98/bookmark-manager.git
    cd bookmark-manager

## Usage

.env.local

    NEXT_PUBLIC_SUPABASE_URL = NEXT_PUBLIC_SUPABASE_URL
    NEXT_PUBLIC_SUPABASE_ANON_KEY = NEXT_PUBLIC_SUPABASE_ANON_KEY

To run the app

    yarn install
    yarn dev
